Usage: "numero" [options] Dates.csv

Options:
  -h [ --help ]                    produce help message
  -n [ --output-lines ] arg (=100) limit number of holidays in the output to 
                                   'arg' lines
  --time-zone arg                  specify output time zone (see README for 
                                   details)

Dates.csv: input dates in CSV (Comma Delimited) format as per README
